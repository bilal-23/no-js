How It Works

This dropdown menu is built using only HTML and CSS, with no JavaScript required. It uses the CSS :hover and :focus-within pseudo-classes on desktop, and leverages the CSS checkbox hack for mobile responsiveness. On mobile devices, this allows users to tap to open/close submenus without requiring JavaScript.

<!-- Simple dropdown example --> <div class="dropdown"> <button class="dropdown-toggle" id="dropdown-toggle" aria-haspopup="true" aria-expanded="false"> Settings <span class="caret">▼</span> </button> <ul class="dropdown-menu" role="menu" aria-labelledby="dropdown-toggle"> <li><a href="#" class="dropdown-item" role="menuitem">Profile</a></li> <li><a href="#" class="dropdown-item" role="menuitem">Preferences</a></li> </ul> </div> <!-- Responsive nested dropdown example --> <nav class="nav-dropdown"> <ul class="nav-menu" role="menubar"> <li class="nav-item has-dropdown" role="none"> <input type="checkbox" id="products-toggle" class="dropdown-checkbox"> <label for="products-toggle" class="dropdown-toggle-label mobile-only"></label> <a href="#" class="nav-link" role="menuitem" aria-haspopup="true" aria-expanded="false"> Products <span class="caret">▼</span> </a> <ul class="nav-dropdown-menu" role="menu"> <li role="none"><a href="#" class="dropdown-item" role="menuitem">Item 1</a></li> </ul> </li> </ul> </nav>

Accessibility Features

The dropdown menu is designed to be accessible, with the following features:

  • Keyboard navigable using Tab and Enter keys
  • Proper focus management with :focus-within
  • Dropdown opens on hover and focus on desktop
  • Touch-friendly interface on mobile with no JavaScript required
  • Semantic HTML structure with proper nesting
  • Fully responsive design that adapts to all screen sizes
  • ARIA attributes for screen readers
