How It Works

This dropdown menu is built using only HTML and CSS, with no JavaScript required. It uses the CSS :hover pseudo-class to trigger the display of the dropdown menu, and the :focus-within pseudo-class to maintain focus for keyboard accessibility.

<button class="dropdown-toggle" id="dropdown-toggle" aria-haspopup="true" aria-expanded="false">Settings <span class="caret">▼</span></button> <ul class="dropdown-menu" role="menu" aria-labelledby="dropdown-toggle"> <li><a href="#" class="dropdown-item" role="menuitem">Profile</a></li> <li><a href="#" class="dropdown-item" role="menuitem">Preferences</a></li> <li><a href="#" class="dropdown-item" role="menuitem">Notifications</a></li> <li class="divider" role="separator"></li> <li><a href="#" class="dropdown-item" role="menuitem">Logout</a></li> </ul>

Accessibility Features

The dropdown menu is designed to be accessible, with the following features:

  • Keyboard navigable using Tab and Enter keys
  • Proper focus management
  • Dropdown opens on hover and focus
  • Semantic HTML structure with proper nesting
  • ARIA attributes for screen readers:
    • role="menu": Identifies the dropdown as a menu
    • role="menuitem": Identifies each item in the menu
    • role="separator": Identifies dividers between menu sections
    • aria-haspopup="true": Indicates the element has a popup
    • aria-expanded: Indicates whether the popup is currently expanded
    • aria-labelledby: Associates the menu with its triggering element
